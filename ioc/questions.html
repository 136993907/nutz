Ioc depose tigger by GC

Top

表现就是标题这句日志了

原因是, "您自己"的代码,创建了一个临时NutIoc容器,然后抛弃掉, 例如

public static Dao dao() {
	IocLoader loader = new JsonLoader("dao.js");
	return new NutIoc().get(Dao.class);
}

解决办法, 全局搜索 new NutIoc, 改成正确的方式, 点我传送门
